# Лабораторная работа 4: Моделирование структуры информационной системы

## Описание

Данная работа содержит UML-диаграммы для модуля аналитики и прогнозирования системы EcoGuardian:
- Диаграмма компонентов (20 компонентов)
- Диаграмма классов (10 классов)

## Файлы проекта

- `ecoguardian_components.puml` - скрипт PlantUML для диаграммы компонентов
- `ecoguardian_classes.puml` - скрипт PlantUML для диаграммы классов
- `report.md` - полный отчет по лабораторной работе
- `PROMPT_AND_CHANGES.md` - описание промптов и изменений (бонусная часть)
- `lab3.md` - техническое задание на разработку модуля

## Генерация диаграмм

### Онлайн-редактор (рекомендуется)
1. Откройте http://www.plantuml.com/plantuml/
2. Скопируйте содержимое `ecoguardian_components.puml` или `ecoguardian_classes.puml`
3. Вставьте в редактор
4. Диаграмма сгенерируется автоматически
5. Сделайте скриншот или скачайте изображение

### Локальная генерация
```bash
plantuml ecoguardian_components.puml
plantuml ecoguardian_classes.puml
```

## Генерация PDF отчета

### Способ 1: Pandoc (рекомендуется)
```bash
pandoc report.md -o report.pdf --pdf-engine=xelatex -V geometry:margin=2cm
```

### Способ 2: Онлайн-конвертеры

Установите расширение "Markdown PDF" и экспортируйте `report.md` в PDF

## Структура диаграмм

### Диаграмма компонентов
- **Интерфейс пользователя**: 3 компонента
- **Модуль аналитики**: 4 сервиса с вложенными компонентами (12 компонентов)
- **Хранилище данных**: 2 базы данных с таблицами (7 компонентов)
- **Всего: 20 компонентов**

### Диаграмма классов

- **Сервисы**: 4 класса
- **Всего: 10 классов**
